# 比特鹰-渠道维护工具

一个用于管理多平台（电鸭社区、V2EX、登链）职位发布的自动化工具。

## 功能特点

- 支持多平台登录状态保持
- 自动填写职位信息
- 现代化的界面设计
- 多线程操作避免界面卡顿

## 环境要求

- Python 3.7+
- Chrome 浏览器
- 相关平台的账号权限

## 安装依赖

```bash
pip install -r requirements.txt
```

## 使用说明

1. 首次使用需要登录各平台
2. 填写职位标题和描述
3. 选择目标平台进行发布
4. 程序会在本地保存登录状态，有效期20天

## 注意事项

- 确保Chrome浏览器已安装
- 首次使用需要登录各平台
- 会在程序目录下生成session文件夹用于保存登录状态

## 开发说明

本项目使用：
- PyQt5 用于GUI界面
- Selenium 用于网页自动化
- webdriver_manager 用于管理Chrome驱动
